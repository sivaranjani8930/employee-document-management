# Centralized Employee Document Management System

A web-based application designed to help organizations securely manage employee documents across multiple stores or branches.

Organizations often face problems such as missing documents, expired licenses, and slow document retrieval during audits. This system provides a centralized platform for HR teams to upload, manage, and track employee documents efficiently.

---

## 🚀 Key Features

✔ **Role-Based Login**  
Secure authentication system with different access levels for administrators and authorized users.

✔ **Employee & Store Mapping**  
Employees can be mapped to specific stores or branches for better organizational management.

✔ **Smart Document Upload with Metadata**  
Upload employee documents with additional metadata such as document type, issue date, and expiry date.

✔ **Advanced Search & Document Retrieval**  
Quickly search and retrieve documents using filters like employee name, document type, or store.

✔ **Document Preview & Secure Download**  
Preview documents directly in the system and download them securely when required.

✔ **Version History Management**  
Maintain historical versions of documents whenever updates are made.

✔ **Expired Document Update System**  
Easily replace expired documents with new records while maintaining history.

✔ **Document Expiry Tracking System**  
Automatically track document expiry dates to ensure timely updates and compliance.

✔ **Real-Time Compliance Dashboard**  
Dashboard showing overall document compliance status, expired documents, and missing records across the organization.

✔ **Store-Wise Employee Document Tracking**  
Track employee documents by store location and quickly identify missing or expired documents for each branch.

✔ **Letter Template & Printing System**  
Generate employee-related letters using predefined templates and print them directly from the system.

## 💡 Problem Solved

Organizations managing multiple employees and stores often struggle with:

- Missing employee documents
- Expired licenses or certifications
- Slow document retrieval during audits
- Lack of centralized compliance monitoring

This system centralizes document management, making it easier for HR teams to store, track, and retrieve employee documents quickly and securely.
